# Snake Game (C++)

A beginner-friendly console **Snake** game written in **C++**.

## Features
- ASCII grid display
- Eat fruit (F) to grow your tail
- Score increases as you eat fruit
- Wall + self-collision = game over
- Controls: **W/A/S/D** (and **Q** to quit)
